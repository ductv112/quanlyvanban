- Convert links to ReactFlow edges: `{ id: link.id.toString(), source: link.from_step_id.toString(), target: link.to_step_id.toString(), style: { stroke: '#64748B' }, markerEnd: { type: 'arrowclosed' } }`

- `onNodesDelete`: For each deleted node: DELETE `/quan-tri/quy-trinh/steps/${stepId}` + Modal.confirm if node has connections
- `onEdgesDelete`: DELETE `/quan-tri/quy-trinh/step-links/${linkId}`
- `onNodeDragStop`: Update node position via PUT `/quan-tri/quy-trinh/steps/${stepId}` with new position_x, position_y
