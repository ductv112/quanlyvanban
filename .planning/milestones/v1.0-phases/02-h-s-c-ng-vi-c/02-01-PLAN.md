**Tables to create:**

1. `edoc.doc_flows` — id SERIAL PRIMARY KEY, unit_id INT NOT NULL REFERENCES departments(id), name VARCHAR(500) NOT NULL, version VARCHAR(50), doc_field_id INT REFERENCES doc_fields(id), is_active BOOLEAN DEFAULT true, created_by INT REFERENCES staff(id), created_at TIMESTAMPTZ DEFAULT NOW(), updated_at TIMESTAMPTZ DEFAULT NOW(). Add UNIQUE(unit_id, name, version).

2. `edoc.doc_flow_steps` — id SERIAL PRIMARY KEY, flow_id INT NOT NULL REFERENCES doc_flows(id) ON DELETE CASCADE, step_name VARCHAR(500) NOT NULL, step_order INT DEFAULT 0, step_type VARCHAR(50) NOT NULL DEFAULT 'process' (values: 'start', 'process', 'end'), allow_sign BOOLEAN DEFAULT false, deadline_days INT DEFAULT 0, position_x FLOAT DEFAULT 0, position_y FLOAT DEFAULT 0, created_at TIMESTAMPTZ DEFAULT NOW().

3. `edoc.doc_flow_step_links` — id SERIAL PRIMARY KEY, from_step_id INT NOT NULL REFERENCES doc_flow_steps(id) ON DELETE CASCADE, to_step_id INT NOT NULL REFERENCES doc_flow_steps(id) ON DELETE CASCADE, created_at TIMESTAMPTZ DEFAULT NOW(). UNIQUE(from_step_id, to_step_id).

4. `edoc.doc_flow_step_staff` — id SERIAL PRIMARY KEY, step_id INT NOT NULL REFERENCES doc_flow_steps(id) ON DELETE CASCADE, staff_id INT NOT NULL REFERENCES staff(id), created_at TIMESTAMPTZ DEFAULT NOW(). UNIQUE(step_id, staff_id).

**Triggers:** `trg_doc_flows_updated_at` on doc_flows.

17. `edoc.fn_report_handling_by_assigner(p_unit_id INT, p_from_date TIMESTAMPTZ, p_to_date TIMESTAMPTZ)` — Returns TABLE(staff_id INT, staff_name TEXT, department_name TEXT, total BIGINT, completed BIGINT, in_progress BIGINT, overdue BIGINT, completion_rate NUMERIC). Group by created_by.
