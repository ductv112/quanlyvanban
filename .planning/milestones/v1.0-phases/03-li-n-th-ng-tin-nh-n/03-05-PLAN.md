**Socket.IO client: `lib/socket.ts`** (per D-21, D-22)

```typescript
import { io, Socket } from 'socket.io-client';
```

Module-level `let socket: Socket | null = null;`

Export `initSocket(token: string): Socket`:
- If socket already connected, return it.
- Create: `io(process.env.NEXT_PUBLIC_API_URL || 'http://localhost:4000', { auth: { token }, autoConnect: true, reconnection: true, reconnectionDelay: 3000 })`.
- Store in module variable.
- Return socket.

Export `getSocket(): Socket | null` — returns current socket.

Export `disconnectSocket(): void` — if socket exists, disconnect and set null.

Export `SOCKET_EVENTS` constant (same as backend):
```typescript
export const SOCKET_EVENTS = {
  NEW_DOCUMENT: 'new_document',
  NEW_MESSAGE: 'new_message',
  NEW_NOTIFICATION: 'new_notification',
  DOC_STATUS_CHANGED: 'doc_status_changed',
} as const;
```
