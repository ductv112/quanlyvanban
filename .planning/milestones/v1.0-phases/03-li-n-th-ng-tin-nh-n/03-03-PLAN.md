**Socket.IO module: `lib/socket.ts`** (per D-18, D-19, D-20)

```typescript
import { Server as SocketIOServer } from 'socket.io';
import type { Server as HTTPServer } from 'http';
import { verifyAccessToken } from './auth/jwt.js';
```

Export `initSocket(httpServer: HTTPServer): SocketIOServer`:
- Create Socket.IO server with cors origin from process.env.CORS_ORIGIN || 'http://localhost:3000', credentials true.
- Authentication middleware: extract token from `socket.handshake.auth.token` or `socket.handshake.headers.authorization` (Bearer prefix). Call `verifyAccessToken(token)`. If valid, attach `socket.data.user = payload` (staffId, unitId, roles). If invalid, call `next(new Error('Authentication error'))`.
- On connection: join room `user_${socket.data.user.staffId}`. Log connection with pino.
- On disconnect: log disconnection.
- Store io instance in module-level variable.

Export `getIO(): SocketIOServer` — returns the stored io instance (throws if not initialized).

Export `emitToUser(staffId: number, event: string, data: unknown)` — convenience: `getIO().to('user_' + staffId).emit(event, data)`.

Export `emitToUsers(staffIds: number[], event: string, data: unknown)` — emits to multiple user rooms.
