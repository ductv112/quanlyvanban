Create migration file `013_sprint8_messages_notices.sql` in edoc schema:

**Table: edoc.messages**
Columns: id BIGSERIAL PK, from_staff_id INT NOT NULL REFERENCES public.staff(id), subject VARCHAR(200) NOT NULL, content TEXT NOT NULL, parent_id BIGINT REFERENCES edoc.messages(id) (for threads — NULL = original message), created_at TIMESTAMP DEFAULT NOW().
Index: idx_messages_from_staff, idx_messages_parent_id.

**Table: edoc.message_recipients**
Columns: id BIGSERIAL PK, message_id BIGINT NOT NULL REFERENCES edoc.messages(id) ON DELETE CASCADE, staff_id INT NOT NULL REFERENCES public.staff(id), is_read BOOLEAN DEFAULT FALSE, read_at TIMESTAMP, is_deleted BOOLEAN DEFAULT FALSE, deleted_at TIMESTAMP.
Index: idx_msg_recipients_staff_id, idx_msg_recipients_message_id.
Unique: uq_msg_recipients_message_staff ON (message_id, staff_id).

**Table: edoc.notices**
Columns: id BIGSERIAL PK, unit_id INT REFERENCES public.departments(id), title VARCHAR(300) NOT NULL, content TEXT NOT NULL, notice_type VARCHAR(50) DEFAULT 'system', created_by INT REFERENCES public.staff(id), created_at TIMESTAMP DEFAULT NOW().
Index: idx_notices_unit_id, idx_notices_created_at.

**Table: edoc.notice_reads**
Columns: id BIGSERIAL PK, notice_id BIGINT NOT NULL REFERENCES edoc.notices(id) ON DELETE CASCADE, staff_id INT NOT NULL REFERENCES public.staff(id), read_at TIMESTAMP DEFAULT NOW().
Unique: uq_notice_reads_notice_staff ON (notice_id, staff_id).

**Message SPs (per D-09 through D-12):**
1. `edoc.fn_message_get_inbox(p_staff_id INT, p_keyword TEXT, p_page INT, p_page_size INT)` — returns messages where staff is recipient, not deleted. Join staff for sender name. Include is_read, total_count. Order by created_at DESC.
2. `edoc.fn_message_get_sent(p_staff_id INT, p_keyword TEXT, p_page INT, p_page_size INT)` — returns messages sent by staff. Include recipient names (array_agg).
3. `edoc.fn_message_get_trash(p_staff_id INT, p_page INT, p_page_size INT)` — returns deleted messages for staff.
4. `edoc.fn_message_get_by_id(p_id BIGINT, p_staff_id INT)` — returns message detail + auto-marks as read (UPDATE message_recipients SET is_read=TRUE, read_at=NOW()). Include sender name, recipient names.
5. `edoc.fn_message_create(p_from_staff_id INT, p_to_staff_ids INT[], p_subject VARCHAR, p_content TEXT, p_parent_id BIGINT)` — insert message + insert message_recipients for each staff_id in array. Returns TABLE(success BOOLEAN, message TEXT, id BIGINT).
6. `edoc.fn_message_reply(p_message_id BIGINT, p_staff_id INT, p_content TEXT)` — creates a new message with parent_id = p_message_id, adds all original recipients + sender as recipients. Returns TABLE(success BOOLEAN, message TEXT, id BIGINT).
7. `edoc.fn_message_delete(p_id BIGINT, p_staff_id INT)` — soft delete: UPDATE message_recipients SET is_deleted=TRUE. Returns TABLE(success BOOLEAN, message TEXT).
8. `edoc.fn_message_count_unread(p_staff_id INT)` — returns TABLE(count BIGINT). Count where is_read=FALSE AND is_deleted=FALSE.

**Notice SPs (per D-14 through D-17):**
9. `edoc.fn_notice_get_list(p_unit_id INT, p_staff_id INT, p_is_read TEXT, p_page INT, p_page_size INT)` — returns notices with is_read flag (LEFT JOIN notice_reads). Filter by unit_id, optional is_read filter ('true'/'false'/NULL for all). Paginate.
10. `edoc.fn_notice_create(p_unit_id INT, p_title VARCHAR, p_content TEXT, p_notice_type VARCHAR, p_created_by INT)` — insert notice. Returns TABLE(success BOOLEAN, message TEXT, id BIGINT).
11. `edoc.fn_notice_mark_read(p_notice_id BIGINT, p_staff_id INT)` — INSERT INTO notice_reads ON CONFLICT DO NOTHING. Returns TABLE(success BOOLEAN, message TEXT).
12. `edoc.fn_notice_mark_all_read(p_staff_id INT, p_unit_id INT)` — mark all unread notices as read for this staff. Returns TABLE(success BOOLEAN, message TEXT, count BIGINT).
13. `edoc.fn_notice_count_unread(p_staff_id INT)` — returns TABLE(count BIGINT). Count notices not in notice_reads for this staff.
